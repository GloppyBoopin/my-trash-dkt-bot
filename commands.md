# Commands

## Key 
| Symbol      | Meaning                        |
|-------------|--------------------------------|
| [Argument]  | Argument is not required.      |

## Math
| Commands | Arguments      | Description |
|----------|----------------|-------------|
| *        | Double, Double |             |
| +        | Double, Double |             |
| -        | Double, Double |             |
| /        | Double, Double |             |

## Moderation
| Commands    | Arguments | Description                     |
|-------------|-----------|---------------------------------|
| thanos-snap | Integer   | thanos snap previous N messages |

## Text
| Commands | Arguments     | Description               |
|----------|---------------|---------------------------|
| avatar   | User          | get any user's avatar     |

| say      | Text          | duplicate input into chat |
|          | Integer, Text |                           |
